---

## ⚙️ Instructions

1. Installer Node.js et SASS (si pas déjà fait) :

```bash
npm install -g sass
```

2. Penser à lancer la compilation automatique sass :

```bash
npm run sass
```

Ouvrir index.html dans un navigateur pour voir le site.
