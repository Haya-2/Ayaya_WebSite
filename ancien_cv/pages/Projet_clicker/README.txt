Je suis SIROT Aline, élève en classe de MMI2 de l'année 2021/22.
J'ai développé mon clicker pour qu'il fonctionne sur Chrome, en 720px et 576px. Il fonctionne avec d'autres résolutions mais n'est pas optimisé.

Les Bonus (Nommées Bases) sont déblocables à partir du premier chiffre atteint. Le deuxième chiffre est le nombre additionné au compteur. C'est un bonus constant et cumulable. Par exemple, pour "Base 3", il faut 500 Notes (Points) pour le débloquer. L'avoir débloqué une fois rajoutera 10 Notes par secondes à votre nombre Total de Notes. Pour 5 "Base 3" achetées, cela fera donc (+ 5 * 10) Notes au Total par seconde.


    • Base 1 : 
Pour chaque Achat de la base : Coûte 20 Notes, donne +1 Notes par seconde et +1 Notes par click.

    • Base 2 :
Pour chaque Achat de la base : Coûte 100 Notes, donne +5 Notes par seconde et +2 Notes par click.

    • Base 3 :
Pour chaque Achat de la base : Coûte 500 Notes, donne +10 Notes par seconde et +5 Notes par click.


    • Default Mode :
Si vous quittez ou rechargez la page, ou que c'est votre première venue, le Default Mode sera enclenché. Il est assez lumineux et propose 3 Bases, 2 Compteurs, un DarkMode et un Mode Normal. Les sons joués sont enlevables. 

    • DarkMode : 
Le DarkMode, aussi mode silencieux, est déblocable avec 10000 Notes. Il permet d'assombrir l'écran et d'enlever les animations inutiles de boutons. Il désactive automatiquement le son des clics. 

    • Mode Normal (après au moins un DarkMode) / Mode Apparance :
Le mode Normal reprend les mêmes principes que le Default Mode, à l'exeption de l'image de fond et du Clic. Si le son avait été enlevé en Default Mode, il reste enlevé.

    • Nouvelle Partie :
Réinitialise toutes les valeures. 

    • Le Temps
J'ai décidé de représenter le temps passé sur le jeu. 


    • Bouton "?" :
Lance l'horrible son de Ramuda ("pyon pyon pyon") ainsi qu'une alerte humouristique, comme pour se moquer du joueur un peu trop curieux.

    • Bouton "GoD" :
En plus de faire une références à des jeu de MOBA, le mot GoD fait références au projet Musical Hypnosis Microphone (HypMic), dont le personnage de Ramuda fait parti. 


    • Mode Développeur :
Un bouton +1000 reste accessible dans le code du jeu. Pour le remettre, il faut enlever les sections commentaires situées en dessous du "mode développeur" sur la page HTML.
